Chapter 8. Application Development with Drivers

Overview

  • Importance of client drivers for connecting to Cassandra.
  • Availability of drivers for multiple programming languages.

Legacy Clients

  • Hector
    • Early Java client for Cassandra.
    • Helped avoid writing directly to Thrift API.
  • Astyanax
    • Java client by Netflix.
    • Successor to Hector, transitioned users from Thrift to CQL.
  • Other Legacy Clients
    • Pycassa (Python)
    • Perlcassa (Perl)
    • Helenus (Node.js)
    • Cassandra-Sharp (.NET)
  • Most legacy clients are no longer maintained.

DataStax Java Driver

  • Most popular and feature-rich driver.
  • New features often appear first in the Java driver.
  • Supports connection pooling, JMX integration, and more.

Development Environment Configuration

  • Use Maven or Gradle to manage dependencies.
  • Example Maven dependency configuration:

<dependency>
  <groupId>com.datastax.oss</groupId>
  <artifactId>java-driver-core</artifactId>
  <version>${driver.version}</version>
</dependency>

Connecting to a Cluster

  • Use CqlSession class to connect.
  • Specify contact points and data center.
  • Example code snippets for connecting to a cluster:

CqlSession cqlSession = CqlSession.builder()
    .addContactPoint(new InetSocketAddress("127.0.0.1", 9042))
    .withLocalDatacenter("datacenter1")
    .build();

Statements

  • Types of statements: SimpleStatement, PreparedStatement, BoundStatement.
  • Creating and executing simple and prepared statements.
  • Example code snippets for inserting and querying data:

// SimpleStatement
SimpleStatement reservationInsert = SimpleStatement.builder(
  "INSERT INTO reservations_by_confirmation (confirm_number, hotel_id, start_date, end_date, room_number, guest_id) VALUES (?, ?, ?, ?, ?, ?)")
  .addPositionalValue("RS2G0Z")
  .addPositionalValue("NY456")
  .addPositionalValue("2020-06-08")
  .addPositionalValue("2020-06-10")
  .addPositionalValue(111)
  .addPositionalValue("1b4d86f4-ccff-4256-a63d-45c905df2677")
  .build();
cqlSession.execute(reservationInsert);

// PreparedStatement
PreparedStatement reservationInsertPrepared = cqlSession.prepare(
  "INSERT INTO reservations_by_confirmation (confirm_number, hotel_id, start_date, end_date, room_number, guest_id) VALUES (?, ?, ?, ?, ?, ?)");
BoundStatement reservationInsertBound = reservationInsertPrepared.bind()
  .setString("confirm_number", "RS2G0Z")
  .setString("hotel_id", "NY456")
  .setLocalDate("start_date", LocalDate.parse("2020-06-08"))
  .setLocalDate("end_date", LocalDate.parse("2020-06-10"))
  .setShort("room_number", 111)
  .setUuid("guest_id", UUID.fromString("1b4d86f4-ccff-4256-a63d-45c905df2677"));
cqlSession.execute(reservationInsertBound);

Query Builder

  • Fluent-style API for building queries programmatically.
  • Example code snippets using QueryBuilder:

// Insert statement using QueryBuilder
Insert reservationInsert = insertInto("reservation", "reservations_by_confirmation")
  .value("confirm_number", literal("RS2G0Z"))
  .value("hotel_id", literal("NY456"))
  .value("start_date", literal(LocalDate.parse("2020-06-08")))
  .value("end_date", literal(LocalDate.parse("2020-06-10")))
  .value("room_number", literal(111))
  .value("guest_id", literal(UUID.fromString("1b4d86f4-ccff-4256-a63d-45c905df2677")));
SimpleStatement reservationInsertStatement = reservationInsert.build();
cqlSession.execute(reservationInsertStatement);

// Select statement using QueryBuilder
Select reservationSelect = selectFrom("reservation", "reservations_by_confirmation")
  .all()
  .whereColumn("confirm_number").isEqualTo(literal("RS2G0Z"));
SimpleStatement reservationSelectStatement = reservationSelect.build();
cqlSession.execute(reservationSelectStatement);

Object Mapper

  • Map Java classes to Cassandra tables using annotations.
  • Simplifies interaction with Cassandra.
  • Example code snippets for entity classes and DAOs:
javaCopy code
// Entity class
import com.datastax.oss.driver.api.mapper.annotations.Entity;
import com.datastax.oss.driver.api.mapper.annotations.PartitionKey;

@Entity
public class ReservationsByConfirmation {
  @PartitionKey
  private String confirmNumber;
  private String hotelId;
  private LocalDate startDate;
  private LocalDate endDate;
  private short roomNumber;
  private UUID guestId;
  // getters and setters
}

// DAO interface
import com.datastax.oss.driver.api.mapper.annotations.Dao;
import com.datastax.oss.driver.api.mapper.annotations.Insert;
import com.datastax.oss.driver.api.mapper.annotations.Select;

@Dao
public interface ReservationDao {
  @Select
  ReservationsByConfirmation findByConfirmationNumber(String confirmNumber);
  @Insert
  void save(ReservationsByConfirmation reservation);
}

Asynchronous Execution

  • Use executeAsync() for non-blocking queries.
  • CompletionStage API for handling async responses.
  • Example code snippets for async operations:
javaCopy code
CompletionStage<AsyncResultSet> resultStage = cqlSession.executeAsync(
  SimpleStatement.newInstance("SELECT * FROM reservations_by_confirmation WHERE confirm_number=?", "RS2G0Z"));
resultStage.thenAccept(resultSet -> {
  Row row = resultSet.one();
  if (row != null) {
    System.out.println("Got reservation: " + row.getString("confirm_number"));
  }
});

Driver Configuration

  • File-based configuration using Typesafe Config.
  • Basic and advanced configuration options.
  • Load balancing, retry policies, speculative execution.
  • Example configuration snippets:

datastax-java-driver {
  basic {
    contact-points = ["127.0.0.1:9042"]
    session-keyspace = reservation
  }
  advanced.speculative-execution-policy {
    class = ConstantSpeculativeExecutionPolicy
    max-executions = 3
    delay = 100 milliseconds
  }
}

Other Drivers

  • DataStax Python Driver
    • Installation and basic usage:
pythonCopy code
from cassandra.cluster import Cluster
cluster = Cluster(['127.0.0.1'])
session = cluster.connect('hotel')
session.execute("INSERT INTO hotels (id, name, phone) VALUES (%s, %s, %s)", ('AZ123', 'Super Hotel at WestWorld', '1-888-999-9999'))
  • DataStax Node.js Driver
    • Installation and basic usage:
javascriptCopy code
const cassandra = require('cassandra-driver');
const client = new cassandra.Client({ contactPoints: ['127.0.0.1'], localDataCenter: 'datacenter1', keyspace: 'hotel' });
client.execute('SELECT * FROM hotels WHERE id=?', ['AZ123'], function(err, result) {
  assert.ifError(err);
  console.log('got hotel with name ' + result.rows[0].name);
});
  • DataStax C# Driver
    • Installation and basic usage:
csharpCopy code
var cluster = Cluster.Builder()
  .AddContactPoints("127.0.0.1")
  .Build();
var session = cluster.Connect("hotel");
session.Execute("INSERT INTO hotels (id, name, phone) VALUES ('AZ123', 'Super Hotel at WestWorld', '1-888-999-9999');");
  • GoCQL Driver for Go language.
  • DataStax C/C++ Driver for asynchronous operations.

Stargate: An Open Source Data Gateway

  • Simplifies API frameworks on top of Cassandra.
  • Supports CQL, REST, GraphQL, gRPC, and document-style REST APIs.
  • Can be installed in front of Cassandra and DataStax Enterprise clusters.
